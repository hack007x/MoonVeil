<strong>镜花水月 「MoonVeil」</strong><br>
集成多种攻防技术与检测手段，帮助安全测试人员高效进行信息收集、漏洞挖掘、利用验证，提升渗透测试的自动化与体系化水平，为安全团队快速发现潜在与攻击面提供充分支持。

<strong>「主要功能」</strong>：端口扫描、指纹存活、域名枚举、POC探测、密码爆破、目录遍历、网络爬虫、空间测绘、请求发包、socks5代理、红队辅助等。
<img width="1433" height="845" alt="image" src="https://github.com/user-attachments/assets/405b60ad-4794-4121-ad7e-a979efd53fca" />

<strong>端口扫描</strong><br>
<span style="font-size:14px; color:#666;">支持C短、单个、批量进行扫描。端口预设 线程、代理通过socks5代理池进行扫描,需要先开启代理设置监听,目前爆破-SYN-UPD模式尚未公开</span>
